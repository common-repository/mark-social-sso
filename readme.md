== Description ==

第一个版本没有做用户绑定功能！用户用不同的社交媒体登陆之后，会自动的为用户创建一个新用户并登陆。 新创建的用户仅用最基本的订阅者权限。

看看有多少人使用的吧，第二个版本会考虑实现多平台用户合并及绑定功能。

== Installation ==

1. 上传到 `/wp-content/plugins/` 目录
2. 在后台插件菜单激活该插件
3. 在wordpress网站后台菜单进行设置： 设置 -> 媒体登陆设置
4. 搞不定，可以加开发者微信: markchenlife (注明来意)
